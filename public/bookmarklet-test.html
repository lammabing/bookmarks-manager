Bookmarklet Test Page

This is a test page to verify that the bookmarklet functionality works correctly.

To test:

  1. Make sure your bookmarks-manager is running on http://localhost:5170
  2. Install the bookmarklet by creating a bookmark with the code from src/bookmarklet.min.js
  3. Click the bookmarklet while on this page
  4. Verify that a new window opens with the form pre-filled with this page's information
